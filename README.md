# TownyPlusPlus
The [TownyPlusPlus](https://github.com/ZombneYT/TownyPlusPlus) adds an addition to everything for [Towny](https://github.com/TownyAdvanced/Towny)!
- war system
- town inventory
- town builds
- town alliance
- town form
- town religion
- town stock
- town wonders
- town century
- town pact
  
(The same thing will happen for the nation)
